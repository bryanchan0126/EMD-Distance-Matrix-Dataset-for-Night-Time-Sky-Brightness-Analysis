**Overview**
This repository contains the complete Earth Mover's Distance (EMD) distance matrix and supporting datasets used in the undergraduate thesis:

"Using AllSky Cameras to Monitor Light Pollution and Its Effects on Green Sea Turtle (Chelonia mydas) Nesting Behavior"

The dataset was generated from keograms produced by AllSky cameras deployed at three study sites in Malaysia:

Chagar Hutang Turtle Sanctuary, Terengganu
Kerachut Beach, Penang
Universiti Sains Malaysia (USM), Penang
**Purpose**
The full EMD distance matrix is provided in this repository because its size makes it difficult to present clearly in printed thesis copies and presentation slides. The matrix was used to quantify similarity between keograms and to support clustering and visualization analyses.
**Analysis**
Earth Mover's Distance (EMD) was applied to compare the visual similarity of keograms generated from night-time sky observations. Lower EMD values indicate greater similarity between keograms, whereas higher values indicate greater differences in sky brightness patterns.
